### Where feedback comes from

Give feedback for AI

1. What appears in browser
2. From devtool
   -  Console
   -  Elements (HTML tags)
   -  Application (cookie, token, local storage)
   -  Network (fetch / xhr / ajax / websocket / server side event(SSE))
3. Terminal (ex: NodeJS server port 3000)
4. Server log (nginx / railway app)
   
### Best start with good foundation knowledge

The more knowledge you have, the better you can prompt, the better result AI will produce
